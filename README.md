# 🐝 BeeHunter
Gamified treasure hunt app to discover Bari, Italy.
## Quick Start
```bash
npm install
npm run dev
## Tech Stack

- React Native 0.81
- Expo 54
- TypeScript
- Tailwind CSS (NativeWind)
- React Context (i18n)

## Features

- 6 main screens (Home, Explore, Partners, Badges, Leaderboard, Profile)
- Multilingual support (Italian & English)
- Gamification system (points, badges, levels)
- Partner integration with discounts
- Light/Dark theme support
